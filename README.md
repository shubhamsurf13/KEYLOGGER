# KEYLOGGER

# 🔑 Keylogger GUI - Python

A simple graphical keylogger built using Python.
This project demonstrates real-time keystroke capturing and display using a GUI interface.

---

## 🚀 Features

* Real-time keystroke logging
* GUI-based interface using CustomTkinter
* Multi-threaded execution (non-blocking UI)
* Displays both normal and special keys

---

## 🛠️ Technologies Used

* Python 3
* CustomTkinter
* Pynput (for keyboard listener)
* Threading

## ⚙️ Installation

### 1. Create Virtual Environment

```id="x6bq7d"
python3 -m venv myenv
source myenv/bin/activate
```

### 2. Install Dependencies

```id="f2p7t9"
pip install customtkinter pynput
```

## ▶️ Usage

Run the program:

```id="wzv3o2"
python keylogger_gui.py
```

Click on **"Start Keylogger"** to begin capturing keystrokes.
